# 🔢 Numexa — Scientific Calculator Discord Bot

Numexa is a powerful and modern **scientific calculator bot for Discord**, built to help students, educators, and math enthusiasts solve problems directly inside Discord.

---

## ✨ Features

- 🧮 Basic & Scientific Calculations  
- 📐 Trigonometry with **Degree / Radian** modes  
- 📘 Differentiation & Integration  
- 🧠 Differential Equation Solver  
- 🔢 Text Count Utility (characters, words, digits, spaces)  
- 🖥️ Interactive Calculator UI (Buttons)  
- ⚡ Prefix & Slash Commands  
- 🔐 Secure Token Handling (Environment Variables)  
- ☁️ 24/7 Hosting Ready (Railway)
